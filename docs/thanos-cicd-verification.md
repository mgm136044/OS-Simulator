## 7. 남은 관찰 사항 (수정 불필요)

### 7.1 RR에서 J1 연속 실행

재큐잉→도착 순서에 의해 J1이 t=0-2, t=2-3으로 연속 실행된다. RR의 일반적 기대와 다를 수 있으나, J2의 도착 시점(AT=1)이 J1의 첫 퀀텀 도중이고 재큐잉이 도착보다 먼저 처리되므로 규칙상 정당하다. 의사코드 하단 주석에서 이미 설명.

### 7.2 6.3절 공정성 다이어그램의 HRRN 위치

HRRN이 SPN 옆(공정성 낮음)에 배치되어 있으나, HRRN은 Response Ratio로 기아를 완화하므로 실제로는 SPN보다 공정성이 높다. 다이어그램은 개념적 단순화이므로 발표 시 구두 보충 가능.

### 7.3 구동 환경

- **개발:** macOS, Python 3.x + PyQt5
- **제출:** Windows `.exe` (PyInstaller `--onefile --windowed`, Task 25)
- CI/CD 분석 문서는 환경에 무관한 알고리즘 분석이므로 구동 환경의 영향 없음
